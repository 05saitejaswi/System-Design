üî• 1. Remove Element
Problem

Remove all occurrences of val from array in-place.

Solution
def removeElement(nums, val):
    k = 0
    for n in nums:
        if n != val:
            nums[k] = n
            k += 1
    return k

üî• 2. Best Time to Buy and Sell Stock

Find max profit with 1 transaction.

Solution
def maxProfit(prices):
    min_price = float('inf')
    max_profit = 0

    for p in prices:
        min_price = min(min_price, p)
        max_profit = max(max_profit, p - min_price)

    return max_profit

üî• 3. Majority Element

Find element appearing more than n/2 times
(Boyer-Moore Algorithm)

Solution
def majorityElement(nums):
    count = 0
    candidate = None

    for n in nums:
        if count == 0:
            candidate = n
        count += 1 if n == candidate else -1

    return candidate

üî• 4. Remove Duplicates from Sorted Array

Keep unique values in-place.

Solution
def removeDuplicates(nums):
    if not nums:
        return 0
    k = 1

    for i in range(1, len(nums)):
        if nums[i] != nums[i-1]:
            nums[k] = nums[i]
            k += 1
    return k
